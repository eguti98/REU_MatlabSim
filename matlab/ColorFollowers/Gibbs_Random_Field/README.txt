In this folder, Gibbs_Random_Field is the first implementation of the GRF cohesion and segregation
method explained in this video: https://www.youtube.com/watch?v=35pEn_CE0Zw

The implementation is based on an edited version of the object oriented discrete sim devloped
by Nate Pearson (np00004@mix.wvu.edu)

To run, go into Eddited_Copy_of_Nate's_Stuff and use Run_Sim. There are still some bugs in this
version and it was mostly kept to show the progress that was made
